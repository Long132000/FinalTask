Git

  • git add - добавляет содержимое рабочего каталога в индекс для последующего коммита.
  • git status - показывает состояние файлов в рабочем каталоге и индексе.
  • git diff - показывает разницу между рабочей копией и индексом или между коммитами.
  • git commit - сохраняет изменения, добавленные в индекс, в репозиторий.
  • git branch - позволяет работать с ветками: создавать, удалять и перечислять их.
  • git merge - объединяет изменения из одной ветки в другую.
  • git checkout - переключает между ветками или восстанавливает файлы из другой ветки.
  • git log - выводит историю коммитов для текущей ветки.
